### TestVectors ###
Test vectors must be formatted with one line for each measurement, with values in the order { A, B, Cin, S, Cout, Ovfl}. Note: A,B,S are in hex, all others are in binary.
Example: 
0AAAAAAAAAAAAAAA 0555555555555555 1 1000000000000000 0 0
